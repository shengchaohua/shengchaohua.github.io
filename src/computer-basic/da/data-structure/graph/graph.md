一、题目

给出graph为有 N 个节点（编号为0, 1, 2, ..., N-1）的无向连通图。

graph.length = N，且只有节点`i`和`j`连通时，`j != i`在列表`graph[i]`中恰好出现一次。

返回能够访问所有节点的最短路径的长度。你可以在任一节点开始和停止，也可以多次重访节点，并且可以重用边。
